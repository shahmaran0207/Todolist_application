# 🚀 **React를 통한 할일 작성 페이지 제작**

- **프로젝트 설명**:  
  React를 사용하여 제작된 **Todolist Application**입니다. 할일 추가 및 제거가 가능한 직관적인 페이지입니다.

## 🔍 **Key Features (주요 기능)**  
### ✅ **React로 구현된 주요 기능**  
- **할일 추가**: 할일 입력 칸을 통해 새 할일을 작성할 수 있습니다.  
- **할일 삭제**: 완료된 할일을 삭제 버튼으로 간편하게 제거할 수 있습니다.
